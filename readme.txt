1. Установка зависимостей
Сначала установите необходимые библиотеки:

bash
pip install tensorflow tensorflow-datasets keras matplotlib numpy
2. Сохранение скрипта
Сохраните объединенный код в файл, например train_flowers_eanet.py

3. Запуск обучения
bash
python train_flowers_eanet.py
4. Что происходит во время выполнения:
Этап 1: Загрузка данных
Автоматически загружается датасет tf_flowers (около 3,670 изображений цветов 5 видов)

Данные разделяются: 85% на обучение, 15% на валидацию

Этап 2: Визуализация
Сначала показываются 9 примеров из датасета

Потом показывается батч из 25 обработанных изображений

Этап 3: Обучение
Модель EANet обучается на цветах в течение 50 эпох

На экране будет отображаться прогресс обучения:

text
Epoch 1/50
49/49 [==============================] - ... - loss: ... - accuracy: ... - val_loss: ... - val_accuracy: ...
Этап 4: Результаты
После обучения показываются графики потерь и точности

Выводится финальная точность на валидационной выборке

5. Ожидаемые результаты
Точность: примерно 50-70% (для небольшой модели на небольшом датасете)

Время обучения: ~30-60 минут на CPU, ~10-15 минут на GPU

6. Возможные проблемы и решения
Если нет GPU:

bash
# Укажите использовать CPU
export CUDA_VISIBLE_DEVICES=""
Если мало памяти:

Уменьшите BATCH_SIZE до 32

Уменьшите patch_size до 8

Если хотите прервать обучение:

Нажмите Ctrl+C

7. Проверка результатов
После обучения модель можно использовать для предсказаний:

python
# Добавьте в конец скрипта
def predict_flower(model, image_path):
    img = tf.io.read_file(image_path)
    img = tf.image.decode_jpeg(img, channels=3)
    img = tf.image.resize(img, [224, 224])
    img = tf.expand_dims(img, 0) / 255.0

    prediction = model.predict(img)
    class_names = ['daisy', 'dandelion', 'roses', 'sunflowers', 'tulips']
    return class_names[tf.argmax(prediction[0])]
